# Car-Racing-Game
A Car Racing Game Based On Native Javascript.

Functions:

- keyDown() 敲击向下方向键时取消默认动作

- keyUp() 敲击向上方向键时取消默认动作

- isCollide() 判断是否相撞

- moveLines() 道路中间线的移动

- endGame() 游戏结束

- moveCar() 赛车移动及判断

- gamePlay() 控制游戏中主体的基本效果和分数记录

- Reset()  重置最高分

- start() 初始化
